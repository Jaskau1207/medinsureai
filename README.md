# MedInsureAI
MedinsureAI is a web app that helps in predicting our Monthly Medical Insurance cost based on our lifestyle choices along with predicting health risks such as diabetes, hypertension and calculating health score and providing suggestions for lifestyle changes and some health insurance plans.
# Features
- Predicting monthly health insurance cost based on lifestyle choices
- Predicting health risk such as diabetes and hypertension based on their lifestyle choices
- Presonalized health score
- Insurance plan recommendations
- Lifestyle planner suggestions
# Tech Stack
- Python (Flask, Pandas, Scikit-learn)
- HTML, CSS, JavaScript
- Machine Learning models for prediction
- Google colab for model training
# About
This project was developed as a part of my 6 weeks (June'25 - July'25) training to implement my learnings. It's a flask based web app built as an outcome of my learnings (Python, Machine Learning, Flask). Though very messy, it demonstrates my learnings from that period.
